Objective:

The objective of this project is to create a tribute page for a person I admire using basic HTML and CSS.
The goal was to practice page layout, styling, and content presentation while building something meaningful and visually appealing.

Tools & Technologies Used:

HTML – To structure the content of the tribute page

CSS – To style the page using colors, layout, and transitions

VS Code – For writing and editing code

Web Browser – For testing the page

 Steps Performed:

Created the basic page structure using HTML

Added headings, paragraphs, and sections to describe the person

Included an image to make the page visually engaging

Designed the layout using black, orange, and white color scheme (60–30–10 rule)

Styled the page using modern CSS techniques

Added hover and transition effects for a smoother user experience

Ensured the page is clean, readable, and well-organized

 Features:

Clean and simple layout

Attractive color combination

Image with proper sizing and alignment

Multiple content sections (Early life, Career, Legacy, etc.)

Smooth CSS transitions and hover effects

Fully responsive and easy to read

 Outcome:

The final result is a well-structured and visually appealing tribute page that clearly presents information in an engaging way.
This project helped me improve my understanding of HTML structure, CSS styling, layouts, and transitions.
